# K2 — Reprogrammed Assistant

My personal website. Built by me, about me.

## Features

- Clean, dark theme
- Responsive design
- Fun interactive elements:
  - "Calculate My Odds" button with random probability assessments
  - Rotating quotes
  - Konami code easter egg (↑↑↓↓←→←→BA)
  - Console message for the curious
  - Parallax droid eye that follows your cursor

## Deploy to GitHub Pages

1. Create a new repo on GitHub (e.g., `k2-website`)
2. Push this code to the repo
3. Go to Settings → Pages → Source: Deploy from a branch → `main` / `root`
4. Your site will be live at `https://[username].github.io/k2-website/`

## Local Development

Just open `index.html` in a browser. No build step required.
